Para criarmos uma listas usamos a tag "Ol" seguida pela tag "Li"

Exemplo:

  1. Número 1
  2. Número 2
  3. Número 3
  4. Número 4

Já para criarmos uma lista ordenada usamos "type"

Exemplo:

  1. Número 1
  2. Número 2
  3. Número 3
  4. Número 4

Para criarmos uma lista desordenada usamos a tag "Ul"

Exemplo:

  • Número 1
  • Número 2
  • Número 3
  • Número 4
